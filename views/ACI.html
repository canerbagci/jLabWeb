ACI: Aho-Corasick Exact Peptide Mapper

Keywords: Proteogenomics, Exact string matching

ACI is a tool that allows matching of multiple peptides to a protein or genome (translated on the fly) database in one pass.

Console application

The console implementation below is intended to be incorporated into computational pipelines or run on the terminal or console.

Requirements

The current release of ACI is compiled on Java 1.8 JDK.
 

Download

Scenario 1 tests whether peptides can be retrieved when they are at the beginning, in the middle or at the end of a sequence for each reading frame.
Scenario 2 tests whether the query peptides are actually matched to all expected locations in the sequence and whether the locations are properly reported.
Scenario 3 tests whether queries which contain shared prefixes, infixes, or suffixes are properly constructed into a query tree, and are mapped to the sequence correctly.
Scenario 4 tests whether overlapping results which can occur with low complex or repetitive peptide sequences are properly mapped to the sequence multiple times for all respective overlaps.
Scenario 5 is a crude test of the scalability of the implementation and tests whether 100000 distinct randomly generated queries can be mapped to a concatenation of the queries as a sequence file.
